# Job Board Platform

## Features
- JWT Authentication
- Role-Based Authorization
- Candidate Profiles
- Resume Upload
- Job Search & Filters
- Job Applications
- Recruiter Dashboard
- Applicant Tracking

## Tech Stack
Frontend:
- Angular 20
- Bootstrap

Backend:
- ASP.NET Core 9
- Entity Framework Core
- SQL Server

Security:
- JWT Authentication
- Route Guards
- HTTP Interceptors
